应用概述

OBox MyCloud是一款私人VPN管理应用，让您能够：

多云部署

在多个云服务商上部署VPN服务器（DigitalOcean、Google Cloud、AWS）

用户管理

创建和管理基于时长和流量限制的VPN用户

安全共享

安全地共享VPN配置

性能监控

监控服务器性能和用户活动

系统要求

  • iOS 17.0或更高版本
  • 活跃的互联网连接
  • 云服务商账户（DigitalOcean、Google Cloud或AWS）

入门指南

初始设置

1. 下载并安装应用

从App Store下载OBox MyCloud应用并完成安装

2. 启动应用

启动应用并查看欢迎界面，浏览各个标签页以熟悉界面：

  • 云服务商：设置和管理云服务器
  • 仪表板：概览和快速操作
  • 用户管理：创建和管理VPN用户
  • VPN服务器：连接和管理服务器
  • 设置：订阅和应用偏好设置

3. 首次配置

  1. 从云服务商标签页开始
  2. 选择您首选的云服务商
  3. 按照所选服务商的设置向导操作
  4. 创建您的第一台VPN服务器
  5. 设置您的第一个用户账户

设置云服务商

DigitalOcean设置

步骤1：身份验证

  1. 导航至云服务商标签页
  2. 选择DigitalOcean
  3. 转到身份验证标签
  4. 获取您的API密钥：
    • 访问DigitalOcean控制台
    • 转到API部分
    • 生成新的个人访问令牌
  5. 在应用中输入API密钥
  6. 启用"记住密钥"以便使用
  7. 点击"连接"进行身份验证

步骤2：区域选择

  1. 移至区域标签
  2. 选择最接近您用户的区域
  3. 推荐区域：NYC1、SFO3、SGP1、FRA1、LON1
  4. 选择后点击"继续"

步骤3：Droplet创建

  1. 转到Droplet标签
  2. 点击"创建新Droplet"
  3. 配置基本设置：
    • 名称：使用自动生成的名称或输入自定义名称
    • 大小：选择推荐的VPN大小（≤$6/月）
    • SSH密钥：使用"自动生成密钥"（推荐）
  4. 点击"创建"并等待完成

步骤4：VPN安装

  1. 移至VPN标签
  2. 点击"开始安装"
  3. 等待SSH连接和Outline安装
  4. 提示时保存配置

Google Cloud Platform设置

步骤1：身份验证

  1. 选择Google Cloud服务商
  2. 点击"使用Google登录"
  3. 完成Google身份验证过程
  4. 成功后将显示您的账户信息

步骤2：项目设置

  1. 转到项目标签
  2. 选择以下之一：
    • 从列表中选择现有项目，或
    • 点击"创建新项目"并输入项目名称
  3. 点击"继续到API"
注意： Google Cloud提供免费套餐，包含E2 Micro实例免费使用。

AWS设置

步骤1：身份验证

  1. 选择AWS服务商
  2. 输入您的访问密钥ID和秘密访问密钥
  3. 如需要，启用"记住凭据"
  4. 点击"连接"
安全提示： 建议为OBox MyCloud创建专用的IAM用户，而不是使用根账户凭据。

创建和管理用户

创建新用户

  1. 导航至用户管理标签页
  2. 点击"+"按钮
  3. 填写用户详细信息：
    • 用户名：必填，唯一标识符
    • 邮箱：可选，用于参考
    • 使用期限：设置过期时间（1-12个月或无限制）
    • 流量限制：设置数据配额（1GB-1000GB或无限制）
  4. 查看预览部分以确保准确性
  5. 点击"创建用户"

订阅限制

  • 免费套餐：最多10个用户
  • 高级套餐：50、200或无限用户
  • 达到限制时在设置中升级订阅

管理VPN服务器

服务器列表概览

VPN服务器标签页显示所有已配置的服务器：

  • 连接状态：在线（绿色）、离线（红色）、连接中（橙色）
  • 服务器信息：名称、IP地址、端口、版本
  • 用户数量：每台服务器上的用户数

仪表板概览

自动刷新

  • 仪表板每30秒自动更新
  • 下拉手动刷新
  • 服务器连接的实时状态

故障排除

常见问题和解决方案

连接问题

问题：无法连接到VPN服务器

解决方案：

  1. 检查互联网连接
  2. 在VPN服务器标签中验证服务器状态
  3. 尝试连接到不同的服务器
  4. 重启应用并重试
  5. 检查云服务商账户状态

常见错误消息

  • "订阅限制已达到" - 升级订阅或删除未使用的用户
  • "服务器无法到达" - 检查互联网连接，验证服务器状态
  • "配置无效" - 重新创建服务器或从备份恢复

备份和恢复

  • 定期备份：每周导出数据库
  • 更新前：应用更新前务必备份
  • 多个位置：在多个位置存储备份
  • 测试恢复：定期验证备份完整性

快速参考

基本工作流程

  1. 首次设置：云服务商 → 身份验证 → 创建服务器 → 安装VPN
  2. 添加用户：用户管理 → 创建用户 → 共享配置
  3. 连接VPN：VPN服务器 → 选择服务器 → 连接
  4. 共享访问：用户管理 → 选择用户 → 选择共享方法

支持资源

  • 应用版本：在设置中检查最新版本
  • 系统要求：iOS 17.0+
  • 存储空间：大约需要50MB
  • 网络：稳定的互联网连接至关重要
